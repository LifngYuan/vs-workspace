# 基础语法

## 一、缩进规则

python 是使用缩进来区分不同的代码块，所以对缩进有严格要求。

### 1.代码缩进规则

- 物理行：代码编辑器中显示的代码，每一行是一个物理行。
- 逻辑行：Python解释器对代码进行解释，一个语句是一个逻辑行。
- 缩进规则

  1. 逻辑行的“首行”需要顶格，即无缩进（也就是一份源码的第一个逻辑行）

  2. 相同逻辑层（同一个代码块）保持相同的缩进量

  3. ```":"```标记一个新的逻辑层
  - 增加缩进表示进入下一个代码层
  - 减少缩进表示返回上一个代码层

### 2、缩进量及缩进字符

Python可以使用空格或制表符（tab符）标记缩进。缩进量（字符个数）不限。
